# Cloudflare R2 User Secrets Setup Commands

## Setting your actual R2 credentials using dotnet user-secrets:

# 1. Account ID (from Cloudflare dashboard)
dotnet user-secrets set "S3Settings:AccountId" "YOUR_ACCOUNT_ID" --project Yuzu.Web

# 2. Service URL (R2 endpoint URL)
dotnet user-secrets set "S3Settings:ServiceUrl" "YOUR_ENDPOINT_URL" --project Yuzu.Web

# 3. Access Key (R2 username/access key)
dotnet user-secrets set "S3Settings:AccessKey" "YOUR_ACCESS_KEY" --project Yuzu.Web

# 4. Secret Key (R2 password/secret key)
dotnet user-secrets set "S3Settings:SecretKey" "YOUR_SECRET_KEY" --project Yuzu.Web

# 5. Bucket Name
dotnet user-secrets set "S3Settings:BucketName" "YOUR_BUCKET_NAME" --project Yuzu.Web

## How to check current secrets:
dotnet user-secrets list --project Yuzu.Web

## How to remove a secret (if needed):
# dotnet user-secrets remove "S3Settings:KeyName" --project Yuzu.Web

## Notes:
# - Replace YOUR_* placeholders with your actual R2 credentials
# - The endpoint URL is typically: https://ACCOUNT_ID.r2.cloudflarestorage.com
# - These secrets are stored locally and won't be committed to git
# - User secrets override values in appsettings.json
